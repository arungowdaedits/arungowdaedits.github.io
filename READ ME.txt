# Personal Portfolio Website

This is a responsive personal portfolio website template built using HTML, CSS, and JavaScript.

## Features
- Responsive layout
- Hero section
- About, Services, Portfolio, Contact sections
- Smooth scrolling and animations

## How to Use
1. Download or clone this repository.
2. Open `index.html` in your browser to view the site.
3. Customize the content, styles, and images to match your personal brand.

## Deployment
To host the site using GitHub Pages:
- Push your code to a GitHub repository.
- Go to Settings → Pages → Select `main` branch → Save.
- Your site will be live at `https://yourusername.github.io/repository-name/`.
